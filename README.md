# Legacy Market Client

Клиентское приложение для старых систем Windows, предназначенное для загрузки и установки программ. Проект ориентирован на пользователей систем Windows, XP, Vista и 7.

## Особенности проекта

* **Поддержка старых систем:** Приложение разработано с использованием чистого Win32 API на C++, что обеспечивает минимальные системные требования и совместимость со старыми версиями ОС Windows (2000 и более).
* **Динамическое управление репозиториями:** программа умеет работать с кастомными репозиториями, что позволяет создавать свои.
* **Категоризация софта:** Распределение программ по вкладкам для удобства.
* **Потоковая загрузка:** приложение загружает выбранный файл и запускает его в системе.
* **Интеграция с инсталляторами:** Автоматический запуск скачанных файлов в системе и отслеживание процесса установки. После завершения работы установщика временные файлы полностью очищаются.

## Архитектура базы данных (apps.ini)
Список программ формируется на стороне репозитория в файле INI. Структура предусматривает секцию категорий и отдельные блоки для каждой утилиты.

## Использование INI файла.

### `Категории`

```
[categories]
list=Категория1,Категория2
```
### `Программа`

```
[IDпрограммы]
name=Имя программы
version=версия программы
url=прямая ссылка на скачивание программы
os=Версии Windows
category=Категория
```

### `Пример INI файла`

```
# категории
[categories]
list=Браузеры,Утилиты

# программа из вкладки Утилиты
[7z920]
name=7-Zip 9.20
version=9.20
url=https://raw.githubusercontent.com/nez3r/LegacyMarket/main/storage/7z920.exe
os=Win98,WinXP,WinVista
category=Утилиты

# программа из вкладки Браузеры
[firefox45]
name=Mozilla Firefox ESR (Ru)
version=45.9.0
url=https://raw.githubusercontent.com/nez3r/LegacyMarket/main/storage/Firefox%20Setup%2045.9.0esr-Ru.exe
os=WinXP
category=Браузеры
```
